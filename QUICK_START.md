## ğŸ‘¥ User Roles Explained

### Administrator (Admin)
- Full system access
- Can manage users and change roles
- Can control traffic signals
- Can review violations
- Access to system settings

### Operator
- Can control traffic signals
- Can review and manage violations
- Cannot manage users or system settings

### User (Regular)
- Read-only access to all data
- Can view dashboards and reports
- Cannot control signals or manage violations
